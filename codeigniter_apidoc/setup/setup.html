安装和使用环境

环境：

  • php >= 5.0
  • codeigniter版本(所有版本)

安装：

  • 运行git clone git@github.com:ym1623/codeigniter_apidoc.git 或者直接下载 zip
  • 下载完成将文件夹phpdoc里面放入codeigniter对应的文件夹中.

目录结构

  • phpdoc
    • config
    • apidocs.php [配置文件]
    • controllers
    • example [测试文件目录]
    • doc.php [运行文件]
    • libraries
    • phpdoc [phpdoc核心 插件文件夹]
